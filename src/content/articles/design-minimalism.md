## 什么是极简

极简不是简单，而是精炼。

就像一首绝句，二十个字道尽人间悲欢；就像一幅水墨，几笔勾勒山河万里。

## 极简设计的原则

### 1. 减法思维

> 完美不是加到不能再加，而是减到不能再减。

每一个元素都要问：它存在有必要吗？

### 2. 留白的力量

留白不是浪费空间，而是给内容呼吸的空间。

中国传统绘画讲究"计白当黑"，空白处自有意境。

### 3. 层次感

极简不意味着单调。通过字号、字重、颜色的微妙变化，建立清晰的视觉层次。

## 代码中的极简

```css
/* 好的设计 */
body {
  font-family: Georgia, serif;
  color: #555;
  line-height: 1.8;
}

/* 过度设计 */
body {
  font-family: Georgia, serif;
  color: #555;
  line-height: 1.8;
  text-shadow: 0 0 1px rgba(0,0,0,0.1);
  background-image: url('noise.png');
}
```

## 结语

极简是一种态度，是对本质的追求。

在这个信息爆炸的时代，学会做减法，或许比做加法更重要。
