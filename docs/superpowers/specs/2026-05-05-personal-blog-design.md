# 个人博客设计文档

## 项目概述

一个极简文艺风格的个人综合博客，融合技术、生活与作品展示。

## 技术栈

- **框架**: Astro + Vue 3 (islands 架构)
- **构建**: Astro SSG 静态生成
- **内容**: Astro Content Collections (Markdown/MDX)
- **交互组件**: Vue 3 (按需加载，零 JS 默认)
- **样式**: CSS (无 UI 框架，手写样式保持设计控制力)
- **部署**: 静态托管 (GitHub Pages / Vercel / Netlify)

### Astro 优势

- 静态生成，首屏极速
- 零 JS 默认，Vue 组件按需水合
- 内置 Markdown 内容集合，自动类型推断
- 文件路由 + 灵活的布局系统
- SEO 友好，天然适合博客

## 视觉设计

### 配色方案 — 墨与纸

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景 | #ececf2 | 灰蓝纸张底色 |
| 标题文字 | #2c2c2c | 深墨色 |
| 正文文字 | #555 | 中灰 |
| 辅助文字 | #888 | 浅灰 |
| 分割线 | rgba(0,0,0,0.06) | 极淡灰 |
| 悬停状态 | rgba(0,0,0,0.04) | 微弱底色变化 |

### 字体

- **标题**: Georgia, 'Noto Serif SC', serif — 衬线字体，文学气质
- **正文**: 'Noto Serif SC', Georgia, serif
- **辅助**: -apple-system, sans-serif (小字、标签)

### 设计原则

- 无彩色点缀，纯灰度层次
- 大量留白，呼吸感
- 低对比度，柔和不刺眼
- 动画克制优雅，不花哨

## 页面结构

### 1. 首页 (Home)

**布局**: 全屏居中英雄区

- 占满整个视口高度
- 居中显示：博客名称（大字）+ 个人签名（小字）
- 右上角导航入口（文字链接，无边框按钮）
- **背景**: 书页翻动的动态效果 — 半透明的书本在背景中缓慢翻页，营造文学氛围

**动态背景细节**:
- 使用 CSS 动画或轻量 Canvas 实现
- 书页以极低透明度 (5-10%) 显示，不干扰前景文字
- 翻页速度缓慢（3-5秒一翻），循环播放
- 书页内容可以是抽象的线条或文字纹理
- 整体效果：像有一本无形的书在身后轻轻翻动

### 2. 文章列表 (Articles)

**布局**: 居中单栏

- 顶部：页面标题「文章」
- 筛选标签栏：全部 / 技术 / 随笔 / 设计（标签切换）
- 搜索框：极简样式，底部线条输入框
- 文章列表：卡片式，每张包含
  - 文章标题
  - 发布日期
  - 分类标签
  - 摘要（1-2行）
  - 阅读更多 →
- 分页或无限滚动

### 3. 文章详情 (Article Detail)

**布局**: 居中单栏，最大宽度 720px

- 文章标题（大字）
- 元信息：日期 · 分类 · 阅读时间
- Markdown 渲染内容
- 底部：上一篇 / 下一篇 导航
- 评论区

### 4. 作品集 (Works)

**布局**: 网格展示

- 顶部：页面标题「作品」
- 2列网格布局
- 每个作品卡片：
  - 封面图（或渐变占位）
  - 作品名称
  - 简短描述
  - 技术标签
  - 链接（在线预览 / GitHub）
- 点击展开详情或跳转

### 5. 关于 (About)

**布局**: 居中单栏

- 头像（圆形，灰度滤镜）
- 名字 + 一句话介绍
- 个人简介段落
- 技能/兴趣标签云
- 时间轴：教育经历 / 工作经历 / 项目里程碑
- 联系方式链接

## 导航设计

- **位置**: 右上角
- **样式**: 纯文字链接，无边框、无背景
- **状态**: 当前页面文字加粗或加下划线
- **移动端**: 汉堡菜单，点击展开全屏导航

## 动画与交互

- **页面切换 — 书页开合**:
  - 离开当前页面时：页面像书页一样从右向左合上（3D 翻转，模拟合书）
  - 进入新页面时：新页面像书页一样从左向右翻开（3D 翻转，模拟翻页）
  - 过渡时间 600-800ms，缓动曲线 ease-in-out
  - 使用 CSS 3D transform (rotateY) + perspective 实现
  - 翻页过程中带有轻微的阴影变化，增强立体感
  - 从首页出发时，英雄区的书本背景配合合上，进入内容页
- **滚动**: 平滑滚动
- **悬停**: 微弱的 opacity 变化
- **文章进入**: 从下方轻微上浮 + 淡入
- **首页背景**: 书页翻动循环动画

## 响应式设计

- **桌面**: > 1024px — 完整布局
- **平板**: 768-1024px — 适当缩放
- **手机**: < 768px — 单列，导航变汉堡菜单
